# Home Lab Project

## Overview

After graduating from college, I wanted to continue building my technical skills rather than letting them sit idle while studying for the CompTIA Security+ certification. To gain more hands-on experience with Linux, system administration, and cybersecurity concepts, I decided to repurpose an old laptop into a dedicated home lab environment.

## Hardware

The device I chose for this project was an older Microsoft Surface Laptop that I used during the beginning of my college career. While it was no longer powerful enough for some of my coursework and development tools, it still had plenty of life left in it for a home lab setup.

### Specifications

- Microsoft Surface Laptop Go
- Intel Core i5 Processor
- 4GB RAM
- 64 GB SSD
- Approximately 50GB of available storage

## Planning the Environment

Before installing an operating system, I spent time researching Linux distributions that would best fit my goals. Since I wanted to explore Linux administration, networking, and cybersecurity tools, a server-focused distribution such as Ubuntu Server was initially my top choice.

However, because I was relatively new to Linux and wanted a more user-friendly experience, I decided to use **Linux Mint Cinnamon Edition**. Linux Mint provides a clean desktop environment that feels familiar to Windows users while still offering the flexibility and power of a Linux-based operating system.

### Why Linux Mint?

- Beginner-friendly interface
- Lightweight compared to modern Windows installations
- Large community support
- Strong compatibility with security and networking tools
- Familiar desktop experience for Windows users

## Creating the Installation Media

To install Linux Mint, I used my primary desktop computer to create a bootable USB drive.

### Tools Used

- **Rufus**
- **32GB USB Flash Drive**
- **Linux Mint Cinnamon Edition ISO**

### Process

1. Downloaded the Linux Mint Cinnamon Edition ISO.
2. Downloaded and installed Rufus.
3. Inserted a 32GB USB drive into my desktop.
4. Used Rufus to create a bootable Linux Mint installer.
5. Configured the Surface Laptop to boot from the USB drive.

This allowed me to test Linux Mint and eventually replace the existing Windows installation with a clean Linux environment dedicated to my home lab.
